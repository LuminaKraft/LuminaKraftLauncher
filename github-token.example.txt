# GitHub Token Configuration for Private Repository

Si tu repositorio es PRIVADO, necesitas configurar un token de GitHub:

## 1. Crear el token:
   - Ve a GitHub Settings > Developer settings > Personal access tokens
   - Generate new token (classic)
   - Selecciona scope 'repo' para repositorios privados
   - Copia el token

## 2. Configurar en el proyecto:
   - Crea un archivo .env.local en la raíz del proyecto
   - Agrega: VITE_GITHUB_TOKEN=tu_token_aqui

## 3. Si el repositorio es PÚBLICO:
   - No necesitas token
   - El sistema funcionará automáticamente

## Ejemplo de .env.local:
VITE_GITHUB_TOKEN=ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
